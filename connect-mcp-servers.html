Connecting MCP Servers in VSCode

You currently have 5 MCP servers configured in your settings file, but only 2 are actively connected to your session.

To Connect All MCP Servers:

  1. Close any running MCP server processes:
    • Press Ctrl+Shift+Esc to open Task Manager
    • Look for any running 'node.exe' or 'supabase-mcp-server.exe' processes
    • End these processes if present
  2. In VSCode, reload the window:
    • Press Ctrl+Shift+P to open the command palette
    • Type 'Developer: Reload Window' and select it
  3. Restart the Cline extension:
    • Press Ctrl+Shift+P to open the command palette
    • Type 'Cline: Restart Extension' and select it

Important: After reloading, you'll need to start a new conversation with Cline to access the newly connected MCP servers.
